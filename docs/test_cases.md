# 多领域法律咨询测试用例

用于验证 LangGraph 多域并行检索系统的测试问题集。
按涉及领域数量分类，每个问题标注了预期命中的领域和对应的法律依据。

---

## 配置

### 多域协作

系统通过 `MULTI_DOMAIN_MAX_DOMAINS` 控制多域并行检索的最大领域数。

| 位置 | 说明 |
|------|------|
| `.env` | 设置 `MULTI_DOMAIN_MAX_DOMAINS=N` 覆盖默认值 |
| `app/config.py` | 默认值 `3` |
| `app/classifier.py` | LLM 输出的领域超过上限时，按相关度排序取前 N 个 |
| `app/law_registry.py` | 提示词限制 LLM 最多输出 3 个领域 |

**生效方式**：修改 `.env` 后重启后端服务即可，无需改代码。

**成本考量**：每个领域并行执行一次完整检索（BM25 + 向量 + Rerank），领域越多 token 消耗越高，响应延迟取决于最慢的分支。建议取值 2\~4。

### 三项优化（默认关闭）

| 环境变量 | 默认值 | 说明 |
|----------|--------|------|
| `ENABLE_WEIGHTED_MERGE` | `false` | 多域加权合并：按领域优先级排序结果（如刑事优先于行政） |
| `DOMAIN_PRIORITY_ORDER` | `刑事,行政,治安,监察` | 领域优先级顺序（逗号分隔） |
| `ENABLE_INTELLIGENT_EXPANSION` | `false` | 上下文智能拓展：LLM Sub Agent 判断候选条文相关性 |
| `EXPANSION_DEPTH` | `1` | 拓展深度：0=关闭 / 1=标准 / 2=深度 |
| `ENABLE_SEMANTIC_VERIFICATION` | `true` | 引用语义溯源：标注置信度 + 检测遗漏引用 |
| `ENABLE_CASE_RETRIEVAL` | `true` | 案例检索：刑事类问题展示相似案例参考 |
| `CASE_USE_SEMANTIC` | `true` | 案例语义检索（LanceDB，首次启动自动构建向量库） |
| `CASE_TOP_K` | `3` | 案例检索返回条数 |

案例检索会按领域自动过滤（如治安问题不返回刑事案例），概览/示例类问题跳过案例检索。

四个优化开关独立，可单独开启测试。关闭时行为与原版完全一致。语义溯源和案例检索默认开启。

## 单领域

### 1. 劳动
**问题：** 试用期被无故辞退，公司需要赔偿吗？能赔多少？
**预期领域：** 劳动
**相关法律：** 劳动合同法

### 2. 婚姻
**问题：** 离婚后孩子抚养权怎么判？法院主要考虑哪些因素？
**预期领域：** 婚姻
**相关法律：** 民法典

### 3. 刑事
**问题：** 盗窃罪的立案标准是多少？偷多少钱才构成犯罪？
**预期领域：** 刑事
**相关法律：** 刑法

### 4. 民事诉讼
**问题：** 去法院起诉需要准备哪些材料？诉讼费怎么算？
**预期领域：** 民事诉讼
**相关法律：** 民事诉讼法

### 5. 网络与数据
**问题：** APP 要求读取通讯录才给用，这合法吗？能拒绝吗？
**预期领域：** 网络与数据
**相关法律：** 个人信息保护法

### 6. 税务
**问题：** 个人所得税的专项附加扣除有哪些？怎么申报？
**预期领域：** 税务
**相关法律：** 增值税法、个人所得税法

### 7. 治安
**问题：** 在小区聚众赌博被抓获，会被拘留多少天？罚款多少？
**预期领域：** 治安
**相关法律：** 治安管理处罚法

### 8. 商事
**问题：** 公司商业秘密被竞争对手窃取，能要求多少赔偿？
**预期领域：** 商事
**相关法律：** 反不正当竞争法

## 双领域

### 1. 劳动 + 刑事
**问题：** 公司高管挪用了员工的工资款，员工该怎么维权？能追究刑事责任吗？
**预期领域：** 劳动、刑事
**相关法律：** 劳动合同法、刑法
**说明：** 劳动方面追讨工资，刑事方面挪用资金罪

### 2. 婚姻 + 网络与数据
**问题：** 离婚诉讼中，配偶偷拍的聊天记录和开房记录能作为出轨证据吗？
**预期领域：** 婚姻、网络与数据
**相关法律：** 民法典、个人信息保护法
**说明：** 证据的合法性判断涉及隐私权

### 3. 治安 + 刑事
**问题：** 被人殴打成轻微伤，对方会被治安拘留还是判刑？
**预期领域：** 治安、刑事
**相关法律：** 治安管理处罚法、刑法
**说明：** 轻微伤走治安处罚，轻伤以上走刑事追诉

### 4. 未成年人 + 刑事
**问题：** 15 岁少年在校打架把同学打成重伤，要负刑事责任吗？家长要赔偿吗？
**预期领域：** 未成年人、刑事
**相关法律：** 未成年人保护法、刑法
**说明：** 刑事责任年龄 + 监护人赔偿责任

### 5. 食药安全 + 民事诉讼
**问题：** 在餐厅就餐后食物中毒住院，该怎么起诉索赔？需要哪些证据？
**预期领域：** 食药安全、民事诉讼
**相关法律：** 食品安全法、民事诉讼法
**说明：** 食品安全责任 + 举证与诉讼流程

### 6. 商事 + 民事诉讼
**问题：** 离职员工把客户名单卖给竞争对手，公司怎么起诉要求赔偿？
**预期领域：** 商事、民事诉讼
**相关法律：** 反不正当竞争法、民事诉讼法
**说明：** 商业秘密侵权 + 诉讼管辖与执行

### 7. 婚姻 + 治安
**问题：** 丈夫长期家暴妻子，女方报警后警方会怎么处理？能申请人身保护令吗？
**预期领域：** 婚姻、治安
**相关法律：** 民法典、治安管理处罚法
**说明：** 家暴属于治安违法行为，人身保护令属于婚姻家事程序

### 8. 监察 + 刑事
**问题：** 政府官员收受企业贿赂为对方谋取项目审批便利，会被怎么处分？构成犯罪吗？
**预期领域：** 监察、刑事
**相关法律：** 监察法、刑法
**说明：** 政务处分 + 受贿罪

## 三领域

### 1. 劳动 + 治安 + 刑事
**问题：** 公司领导性骚扰女员工后，以旷工为由将其违法辞退，员工该如何维权？
**预期领域：** 劳动、治安、刑事
**相关法律：** 劳动合同法、治安管理处罚法、刑法
**说明：**
- 劳动：违法解除劳动合同的赔偿
- 治安：性骚扰属于治安违法行为
- 刑事：情节严重的可能构成强制猥亵罪

### 2. 未成年人 + 刑事 + 民事诉讼
**问题：** 14 岁少年校园霸凌致同学重伤，对方家长要求巨额赔偿，该怎么处理？
**预期领域：** 未成年人、刑事、民事诉讼
**相关法律：** 未成年人保护法、刑法、民事诉讼法
**说明：**
- 未成年人：校园欺凌防治
- 刑事：故意伤害罪的刑事责任年龄
- 民事诉讼：监护人代为应诉与赔偿

### 3. 食药安全 + 刑事 + 民事诉讼
**问题：** 商家销售掺假牛肉被查出，消费者集体食物中毒，消费者该怎么维权？
**预期领域：** 食药安全、刑事、民事诉讼
**相关法律：** 食品安全法、刑法、民事诉讼法
**说明：**
- 食药安全：食品安全违法行为
- 刑事：生产、销售伪劣产品罪
- 民事诉讼：集体诉讼与赔偿

### 4. 劳动 + 刑事 + 民事诉讼
**问题：** 老板拖欠工资还威胁恐吓员工，员工该怎么维权？能否同时报警和申请仲裁？
**预期领域：** 劳动、刑事、民事诉讼
**相关法律：** 劳动合同法、刑法、民事诉讼法、仲裁法
**说明：**
- 劳动：拖欠工资仲裁
- 刑事：威胁恐吓可能构成寻衅滋事
- 民事诉讼：仲裁与诉讼的衔接

### 5. 监察 + 刑事 + 行政
**问题：** 行政执法人员收受贿赂后违规发放许可证，相关方该怎么投诉和举报？
**预期领域：** 监察、刑事、行政
**相关法律：** 监察法、刑法、行政复议法
**说明：**
- 监察：公职人员违法违纪调查
- 刑事：受贿罪
- 行政：违法行政许可的撤销与复议

### 6. 劳动 + 未成年人 + 民事诉讼
**问题：** 16 岁少年冒用他人身份证入职后遭遇工伤，公司发现后拒绝赔偿并报警，家长该怎么办？
**预期领域：** 劳动、未成年人、民事诉讼
**相关法律：** 劳动合同法、未成年人保护法、民事诉讼法
**实测分类：** 劳动、未成年人、民事诉讼 — 三域并行检索，成功
**说明：**
- 劳动：工伤认定与赔偿（劳动关系是否成立）
- 未成年人：未成年人劳动权益保护
- 民事诉讼：申请仲裁、起诉举证

## 四领域

> 默认配置下 `MULTI_DOMAIN_MAX_DOMAINS=3`，四领域问题会被截取为相关度最高的 3 个。
> 需要测试完整四域时，设置 `MULTI_DOMAIN_MAX_DOMAINS=4` 并重启服务。

### 1. 食药安全 + 刑事 + 商事 + 民事诉讼
**问题：** 某公司生产销售假药，通过不正当竞争手段抢占正规药企市场，消费者服用后住院，该怎么追究？
**预期领域：** 食药安全、刑事、商事、民事诉讼
**相关法律：** 食品安全法、刑法、反不正当竞争法、民事诉讼法
**说明：**
- 食药安全：假药劣药问题
- 刑事：生产、销售假药罪
- 商事：不正当竞争、市场秩序
- 民事诉讼：消费者集体诉讼索赔

### 2. 监察 + 刑事 + 行政 + 税务
**问题：** 政府税务官员收受贿赂，帮助企业偷税漏税并违规办理退税，企业后续该怎么应对行政调查？
**预期领域：** 监察、刑事、行政、税务
**相关法律：** 监察法、刑法、行政复议法、增值税法
**说明：**
- 监察：公职人员职务违法调查
- 刑事：受贿罪 + 徇私舞弊罪
- 行政：行政复议应对税务处罚
- 税务：偷税漏税的法律责任

### 3. 未成年人 + 刑事 + 治安 + 网络与数据
**问题：** 中学生被同学网络暴力，在网上散布其隐私照片并辱骂威胁，导致受害人精神崩溃，该怎么维权？
**预期领域：** 未成年人、刑事、治安、网络与数据
**相关法律：** 未成年人保护法、刑法、治安管理处罚法、个人信息保护法
**说明：**
- 未成年人：校园欺凌与网络沉迷防治
- 刑事：侮辱诽谤罪（自诉）
- 治安：威胁恐吓的治安处罚
- 网络与数据：个人信息泄露、隐私权侵害

## 案例检索

> CaseMatch 案例库基于 LeCaRD 刑事裁判文书数据集，覆盖盗窃、故意伤害、诈骗、
> 危险驾驶、走私等常见罪名。案例检索作为法律条文检索的**补充参考**，在回答
> 末尾以「相似案例参考」卡片展示（含案例摘要、法院说理、争议焦点）。
>
> 案例检索支持领域过滤：按查询领域（如"治安"）自动匹配案例库中的 `legal_domain`，
> 避免跨领域案例干扰。概览/示例类问题（含"了解""法律规定""示例"等关键词）自动跳过案例检索。

### 1. 盗窃罪 — 量刑参考

**问题：** 入室盗窃价值三万元财物，会被判几年？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 回答末尾出现「相似案例参考」，案例应涉及盗窃罪，含法院量刑说理

### 2. 故意伤害罪 — 轻伤认定

**问题：** 因口角纠纷将人打成轻伤二级，对方不谅解，会怎么判？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 案例应涉及故意伤害 + 轻伤伤情等级 + 量刑考量

### 3. 诈骗罪 — 网络诈骗

**问题：** 在网上发布虚假商品信息骗取多人货款共计五万元，构成什么罪？判多久？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 案例应涉及诈骗罪，可能含电信网络诈骗相关情节

### 4. 危险驾驶罪 — 醉驾

**问题：** 醉酒驾驶血液酒精含量 180mg/100ml，没有造成事故，会怎么处理？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 案例应涉及危险驾驶罪 + 血液酒精含量与量刑的关系

### 5. 掩饰隐瞒犯罪所得罪 — 收赃

**问题：** 明知是盗窃来的手机还以低价收购，会构成犯罪吗？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 案例应涉及掩饰、隐瞒犯罪所得罪 + "明知"的认定标准

### 6. 走私罪 — 走私普通货物

**问题：** 从国外代购奢侈品入境未申报，金额超过十万元，会被追究刑事责任吗？
**预期领域：** 刑事
**相关法律：** 刑法
**验证要点：** 案例应涉及走私普通货物罪 + 偷逃税额计算

### 2. Python 脚本测试（避免 Windows 编码问题）

```python
import json, urllib.request

# 测试多域问题
body = json.dumps({
    "question": "16岁少年冒用他人身份证入职后遭遇工伤，公司发现后拒绝赔偿并报警，家长该怎么办？",
    "session_id": "test_3domain"

req = urllib.request.Request("http://localhost:8080/api/chat/stream", data=body,
    headers={"Content-Type": "application/json"})
resp = urllib.request.urlopen(req, timeout=120)

current_event = None
for raw_line in resp:
    line = raw_line.decode("utf-8").strip()
    if line.startswith("event: "):
        current_event = line[7:]
    elif line.startswith("data: ") and current_event:
        data = json.loads(line[6:])
        print(f"[{current_event}] {json.dumps(data, ensure_ascii=False)}")
        current_event = None
```

```python
# 测试案例检索（仅打印 case_results）
import json, urllib.request

body = json.dumps({

resp = urllib.request.urlopen(req, timeout=120)

current_event = None
for raw_line in resp:
    line = raw_line.decode("utf-8").strip()
    if line.startswith("event: "):
        current_event = line[7:]
    elif line.startswith("data: ") and current_event:
        data = json.loads(line[6:])
        if current_event == "done":
            cases = data.get("case_results", [])
            print(f"[案例检索] 命中 {len(cases)} 条：")
            for c in cases:
                print(f"  - {c.get('title', c.get('case_id', '?'))}")
                print(f"    罪名: {c.get('charges_text', '-')}")
                print(f"    摘要: {(c.get('case_summary', '') or '')[:80]}...")
        current_event = None
```

### 3. 观察服务日志

| 日志标记 | 含义 |
|----------|------|
| `[分类]` | 单域分类结果（单域问题） |
| `[多域分类]` | 多域分类结果及 `is_multi_domain` 标记 |
| `[sub_question]` | 为每个领域生成的子问题 |
| `[retrieve_one_domain]` | 某个领域的并行检索完成 |
| `[merge_contexts]` | 多域检索结果合并去重 |
| `[混合检索]` | BM25 + 向量 RRF 融合结果数 |
| `[Rerank]` | Rerank 精排前后数量 |
| `[流式输出]` | LLM 流式回答开始 |
| `[案例库]` | 案例库加载状态（启动时） |
| `SQLite 已加载` | FTS5 案例库就绪 |
| `LanceDB 已加载` | 语义案例库就绪（首次启动显示"从 SQLite 构建中"） |
| `case_results` | `done` 事件中的 `case_results` 字段，有值即案例检索生效 |
